<h1>Chat App </h1>

-  Tech Stack: Next.js, TypeScript, Tailwind CSS, `UPSTASH` Redis
-  Authentication
-  Real-Time Messaging
-  Responsive UI
-  Light and Dark Mode
-  Notification Sounds
-  Typing Effects
-  Image Uploads
-  Integration with Upstash Redis for Scalable Chat




### Install dependencies

```shell
npm install
```

### Start the app

```shell
npm run dev
```
